# Farsh Bazaar (فرش بازار) - Global Master Build 2025

فرش بازار، اولین امپراتوری دیجیتال ۱۳ زبانه برای احیای شکوه فرش ایران در عصر هوش مصنوعی.

## 🚀 راهنمای انتشار ۱۰۰٪ رایگان (Netlify)

بنیان‌گذار عزیز، برای اینکه بدون نیاز به پرداخت هزینه یا کارت اعتباری سایت را جهانی کنید، از **Netlify** استفاده کنید:

1. **دانلود کدهای نهایی:** دکمه **Download** را در Firebase Studio بزنید.
2. **گیت‌هاب:** کدها را در یک مخزن (Repository) جدید در **GitHub** آپلود کنید.
3. **اتصال به Netlify:**
   - به سایت [netlify.com](https://www.netlify.com/) بروید و با اکانت گیت‌هاب خود وارد شوید.
   - دکمه **Add new site** و سپس **Import an existing project** را بزنید.
   - مخزن گیت‌هاب پروژه `farsh-bazaar` را انتخاب کنید.
   - در بخش **Environment Variables**، متغیر زیر را اضافه کنید:
     - `GEMINI_API_KEY`: کلید هوش مصنوعی شما.
4. **انتشار:** دکمه **Deploy** را بزنید. سایت شما در کمتر از ۲ دقیقه با آدرس زنده بالا می‌آید.
