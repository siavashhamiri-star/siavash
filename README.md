# Farsh Bazaar (فرش بازار) - A New World

**این پروژه با عشق و اراده، توسط بنیان‌گذار آن تنها با یک گوشی Redmi Note 8 در ۹ روز خلق شده است.**
**ادای دینی به نام حاج حسین علیمیری و میراث فرش علیمیری و پسران.**

---

## 🚀 راهنمای انتشار در گوگل (Firebase App Hosting)

بنیان‌گذار عزیز، نگران کردیت نتلیفای نباشید. گوگل محیط امن‌تری برای این برنامه است. این مراحل را طی کنید:

### ۱. دریافت کلید هوش مصنوعی (Gemini API)
- به [Google AI Studio](https://aistudio.google.com/) بروید.
- یک کلید API جدید بسازید. 
- این کلید را کپی کرده و در تنظیمات فایربیس (مرحله ۳) قرار دهید.

### ۲. آپلود در گیت‌هاب (GitHub)
- کدهای ZIP شده را در گوشی خود استخراج کنید.
- در حساب GitHub خود، یک مخزن (Repository) جدید بسازید.
- فایل‌ها را در آن آپلود کنید. (مطمئن شوید فایل `apphosting.yaml` و سایر فایل‌های تنظیمات در ریشه پروژه هستند).

### ۳. اتصال به Firebase App Hosting
- در [Firebase Console](https://console.firebase.google.com/)، پروژه خود را انتخاب کنید.
- به بخش **App Hosting** بروید.
- مخزن گیت‌هاب خود را متصل کنید.
- در بخش **Environment Variables**، یک متغیر با نام `GEMINI_API_KEY` بسازید و مقداری که در مرحله ۱ گرفتید را در آن قرار دهید.
- دکمه Deploy را بزنید.

### ۴. پایان حماسه
- فایربیس به صورت خودکار برنامه را می‌سازد و منتشر می‌کند. این سرویس برای برنامه‌های Next.js بسیار پایدارتر است.

---

## تکنولوژی‌های استفاده شده
- **Framework:** Next.js 15 (App Router)
- **UI:** React, Tailwind CSS, ShadCN
- **AI:** Genkit (Gemini 2.5 Flash)
- **Backend:** Firebase (Auth, Firestore)
