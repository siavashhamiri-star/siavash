
# Farsh Bazaar (فرش بازار) - A New World

**این پروژه با عشق و اراده، توسط بنیان‌گذار آن تنها با یک گوشی Redmi Note 8 در ۹ روز خلق شده است.**

---

## 📱 راهنمای مخصوص موبایل برای انتشار (Mobile Deployment Guide)

بنیان‌گذار گرامی، برای اینکه ۲۰٪ کردیت باقی‌مانده شما هدر نشود، دقیقاً این مراحل را طی کنید:

### ۱. کلید هوش مصنوعی (API Key)
- به [Google AI Studio](https://aistudio.google.com/) بروید.
- یک کلید API بسازید. این کلید برای بخش "داستان‌سرای فرش" ضروری است.

### ۲. آپلود در GitHub (تنها راه مطمئن برای موبایل)
- کدهای ZIP شده را در گوشی خود ذخیره کنید.
- در حساب GitHub خود (از طریق مرورگر گوشی در حالت Desktop Site)، یک مخزن جدید بسازید و فایل‌ها را آپلود کنید.

### ۳. اتصال به Netlify
- وارد Netlify شوید.
- گزینه **Add new site** و سپس **Import from an existing project** را بزنید.
- به GitHub متصل شوید و پروژه را انتخاب کنید.
- **مهم:** نتلیفای به طور خودکار فایل `netlify.toml` را می‌خواند و از ۲۰٪ کردیت شما به بهترین شکل استفاده می‌کند.

### ۴. تنظیم متغیر محیطی
- در پنل Netlify، به بخش **Site settings** > **Environment variables** بروید.
- متغیر `GEMINI_API_KEY` را با مقداری که در مرحله ۱ گرفتید تعریف کنید.

---

## تکنولوژی‌های استفاده شده
- **Framework:** Next.js 15 (App Router)
- **UI:** React, Tailwind CSS, ShadCN
- **AI:** Genkit (Gemini 2.5 Flash)
- **Backend:** Firebase (Auth, Firestore)

این برنامه، ادای دینی است به نام **حاج حسین علیمیری** و میراث **فرش علیمیری و پسران**.
