# Farsh Bazaar (فرش بازار) - Global Master Build 2025

فرش بازار، اولین امپراتوری دیجیتال ۱۳ زبانه برای احیای شکوه فرش ایران در عصر هوش مصنوعی.

## 🚀 راهنمای استقرار و انتشار (Deployment Guide)

بنیان‌گذار عزیز، برای انتشار موفقیت‌آمیز مراحل زیر را دنبال کنید:

1. **دانلود کدهای نهایی:** دکمه **Download** را در بالای صفحه استودیو بزنید.
2. **آپلود در GitHub:** کدها را در یک مخزن (Repository) جدید آپلود کنید.
3. **تشخیص برنچ (Production Branch):** 
   - پس از آپلود، در صفحه اصلی مخزن خود در GitHub، نام شاخه را چک کنید.
   - معمولاً نام آن `main` است. اگر قدیمی‌تر باشد ممکن است `master` باشد. 
   - در پلتفرم‌های انتشار (مثل Cloudflare یا Vercel) دقیقاً همان نامی را وارد کنید که در گیت‌هاب می‌بینید.
4. **تنظیمات Cloudflare/Vercel:**
   - Framework Preset را روی **Next.js** تنظیم کنید.
   - در بخش Environment Variables، کلید `GEMINI_API_KEY` را اضافه کنید.
