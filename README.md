## 🚀 Quick GitHub Connection Guide (For Mobile/Termux)

1. **Download Project:** Click the download button in Firebase Studio.
2. **Extract:** Unzip the `project.zip` file.
3. **Initialize Git:**
   ```bash
   cd project-folder
   git init
   git add .
   git commit -m "Initial launch of Aras Heritage"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

## 🛠 Deployment on Coolify / Self-Hosted

### 1. Build Commands
- **Build Command:** `npm run build`
- **Start Command:** `npm run start`
- **Node.js Version:** `20`

### 2. Environment Variables (CRITICAL)
Set these keys in your Coolify project settings under **Environment Variables**:
- `GEMINI_API_KEY`: Get from [Google AI Studio](https://aistudio.google.com/app/apikey)
- `NEXT_PUBLIC_FIREBASE_API_KEY`: From Firebase Project Settings
- `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN`: From Firebase Project Settings
- `NEXT_PUBLIC_FIREBASE_PROJECT_ID`: From Firebase Project Settings
- `NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET`: From Firebase Project Settings
- `NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID`: From Firebase Project Settings
- `NEXT_PUBLIC_FIREBASE_APP_ID`: From Firebase Project Settings
