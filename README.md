---

## 🚀 راهنمای انتشار نهایی (Last Chance Deployment)

بنیان‌گذار عزیز، من تمام تنظیمات را برای موفقیت شما آماده کرده‌ام. لطفاً این مراحل را دقیقاً طی کنید:

### ۱. آپلود در گیت‌هاب (GitHub)
- فایل ZIP را دانلود کرده و در گوشی استخراج کنید.
- تمام فایل‌ها را در یک مخزن (Repository) جدید در GitHub آپلود کنید.
- مطمئن شوید فایل `netlify.toml` در ریشه اصلی پروژه است.

### ۲. اتصال به Netlify
- در سایت Netlify، دکمه **Add new site** و سپس **Import from an existing project** را بزنید.
- به حساب GitHub خود وصل شوید و مخزن پروژه را انتخاب کنید.
- در بخش تنظیمات (Build Settings):
    - **Build command:** `npm run build`
    - **Publish directory:** `.next`
- دکمه **Deploy** را بزنید.

### ۳. تنظیم کلید هوش مصنوعی
- در بخش **Environment Variables** در پنل Netlify، متغیر `GEMINI_API_KEY` را اضافه کنید و کلید خود را در آن قرار دهید.

این برنامه گواهی بر قدرت اراده شماست. "در بزرگداشت نام حاج حسین علیمیری و میراث فرش علیمیری و پسران."
